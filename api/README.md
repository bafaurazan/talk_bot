# talk_bot

## To run project

- change file ```.env.example``` to ```.env```
- in created```.env```file, in ```TOKEN="DISCORD_TOKEN"``` paste **your discord bot token**

## In terminal

- go to folder

``` shell
cd api
```

- and type

``` shell
poetry install
```

``` shell
poetry run python open.py
```

## If you strugle with python version, type the following

``` shell
poetry env use 3.12
```
